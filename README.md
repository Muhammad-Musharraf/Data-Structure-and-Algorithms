# # Data Structures and Algorithms in C++

This repository provides a comprehensive collection of fundamental data structures and algorithms implemented in C++. It serves as a practical resource for students, developers, and anyone looking to strengthen their understanding of core computer science concepts.

## 📚 Implemented Data Structures & Algorithms

This project covers a wide range of topics, with clear C++ implementations for each.

### Data Structures

*   **Linked Lists**
    *   Singly Linked List
    *   Doubly Linked List
    *   Singly Circular Linked List
    *   Doubly Circular Linked List
*   **Stacks**
    *   Static Stack (Array-based)
    *   Dynamic Stack (Linked List-based)
*   **Queues**
    *   Static Queue (Array-based)
    *   Dynamic Queue (Linked List-based)
*   **Hashing**
    *   Open Hashing (Chaining)
    *   Closed Hashing (Linear & Quadratic Probing)
*   **Trees**
    *   Binary Search Tree (BST)
    *   AVL Tree
    *   Heap (Max Heap)
*   **Graphs**
    *   Adjacency List representation
    *   BFS & DFS Traversals
    *   Cycle & Path Detection

### Algorithms

*   **Searching Algorithms**
    *   Linear Search (including handling repeated values)
    *   Binary Search (including finding first/last occurrences of repeated values)
*   **Sorting Algorithms**
    *   Bubble Sort
    *   Selection Sort
    *   Insertion Sort
    *   Merge Sort
    *   Quick Sort
*   **Expression Evaluation**
    *   Infix to Postfix Conversion
    *   Postfix Expression Evaluation

## 🛠️ Tech Stack

*   **Language:** C++

## 🚀 Getting Started

To use this repository, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Muhammad-Musharraf/Data-Structure-and-Algorithms.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Data-Structure-and-Algorithms
    ```

3.  **Compile and Run a File:**
    Navigate to the specific folder and use a C++ compiler like `g++` to compile and run any of the source files.
    ```bash
    // Example for BubbleSort
    cd Sorting
    g++ BubbleSort.cpp -o BubbleSort
    ./BubbleSort
    ```
